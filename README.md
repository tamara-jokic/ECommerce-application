# ECommerce Application

## Getting Started

These instructions will get you start this project locally.

### Prerequisites

First you have to have Node, Json and npm installed on your computer.

- [Node](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com/)
- [JSON](https://www.npmjs.com/package/json-server)

### Installing

clone https://github.com/tamara-jokic/ECommerce-application.git

```
npm install
```

```
npm start
```

This will start the webpack dev server.
Open http://localhost:8080 to view it in the browser.

### Running Node server

```
node app.js
```

### Running Json server

cd into moc folder, and

```
json-server --watch db.json
```
